# Source
AASM 2017 OSA diagnostic testing guideline

# Intended Use
Used for warning signs and referral prompts only.

# Summary
- OSA suspicion rises when habitual snoring is accompanied by witnessed apneas, gasping or choking awakenings, excessive daytime sleepiness, or other clinically relevant symptoms.
- A user-facing education layer can safely encourage structured symptom tracking and medical follow-up for formal evaluation, but it should not claim that OSA is present or suggest device settings or purchases.
