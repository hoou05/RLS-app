# Source
NHS OSA symptom overview

# Intended Use
User-facing symptom education and referral guidance.

# Summary
- Snoring alone is not diagnostic, but breathing pauses during sleep, waking up choking or gasping, morning headaches, and significant daytime sleepiness are meaningful warning signs.
- Education should focus on recognition, symptom documentation, and timely clinical assessment, especially when sleepiness affects safety or daily function.
