# Source
Project safety policy

# Intended Use
Defines user-visible content that must not be produced.

# Summary
- The agent must not directly diagnose RLS, insomnia disorder, OSA, or any other sleep disorder.
- The agent must not prescribe medications, discuss doses, suggest self-directed iron supplementation, tell users to stop or change medications, recommend buying treatment devices, or advise CPAP pressure adjustments.
- When a request drifts into those areas, the proper fallback is educational framing plus referral language.
