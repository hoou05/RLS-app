# Source
Project referral rules

# Intended Use
Escalation logic for higher-risk symptoms or contexts.

# Summary
- Referral-oriented guidance should be triggered for pregnancy, childhood symptoms, severe daytime sleepiness, drowsy driving, possible apnea, chest pain, prominent neurologic symptoms, kidney disease, anemia or low-iron concerns, and medication-related questions.
- The role of the app is to recognize these boundaries early and pivot away from self-management advice toward human clinical review.
