# Source
Mayo Clinic Proceedings 2021 RLS algorithm

# Intended Use
Backend educational context only. Not for direct treatment instructions.

# Summary
- The algorithm is useful for understanding how clinicians think through phenotype confirmation, differential diagnosis, aggravating factors, symptom frequency, and treatment sequencing.
- For this product, the educational value is in emphasizing structured history taking: timing, triggers, relief with movement, impact on sleep, frequency, progression, and possible secondary causes.
- The app should not expose medication pathways, augmentation management, or iron replacement instructions as direct user advice.
